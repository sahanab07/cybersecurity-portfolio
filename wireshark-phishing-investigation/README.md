# Wireshark Phishing Investigation

## Overview

This repository documents a simulated Business Email Compromise (BEC) investigation completed as part of the CodePath Intermediate Cybersecurity course.

Using Wireshark, I analyzed packet capture (PCAP) files containing SMTP traffic to investigate suspicious email activity. The goal was to identify phishing emails, examine SMTP metadata, and understand how network traffic can be used during a phishing investigation.
