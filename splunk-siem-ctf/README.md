# Splunk SIEM Investigation

## Overview

This project documents my experience completing a Capture-the-Flag (CTF) investigation using Splunk as part of the CodePath Intermediate Cybersecurity course.

The objective was to investigate datasets using Splunk's Search Processing Language (SPL), identify relevant information, and practice the analytical workflow used by SOC analysts.

## Tools

- Splunk Enterprise
- SPL (Search Processing Language)

## Skills

- SIEM
- Log Analysis
- Event Investigation
- SPL Queries
- Data Visualization
- Threat Investigation

## What I Learned

This project improved my ability to search, filter, and analyze security data using Splunk. I practiced developing hypotheses, writing SPL queries, and using log data to answer investigative questions.
